# "Dumbo" 20mm Grenade Launcher

This Addon adds a single fire 20mm Grenade Launcher to [Hideous Destructor](https://codeberg.org/mc776/HideousDestructor).

Requires both [HDCoreLib](https://github.com/HDest-Community/hdest-core-lib) AND [HDBulletLib-Recasted](https://github.com/HDest-Community/HDBulletLib-Recasted)!!!

## Contents

### ''Dumbo'' 20mm Grenade Launcher

    * Can spawn on Rocket Launchers
    * Uses 20mm Grenades
    * Single fire
    * It looks like an elephant :)
    * Fits in a backpack
    * Can be sold by Merchants
    * Price and Spawn weight can be adjusted in the menu
    * Loadout code: DMB
    * Class: VirDumbo
